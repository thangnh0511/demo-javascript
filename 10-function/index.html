CÁCH TRUYỀN GIÁ TRỊ

  • Truyền tham chiếu (địa chỉ của biến): Array, Object
  • Truyền tham trị (giá trị của biến): String, Integer, ....

CONTINUE - BREAK - RETURN

  • RETURN - trả về giá trị, phải tạo biến để trả về
  • CONTINUE - Bỏ qua câu lệnh bên dưới và tiếp tục vòng lặp
  • BREAK - Thoát ra khỏi vòng lặp
